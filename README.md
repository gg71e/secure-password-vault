# secure-password-vault
A secure desktop application built with Python for generating, evaluating, and storing passwords locally.

## Features

* Secure password storage
* Strong password generation
* Password strength evaluation
* Local data storage
* User-friendly graphical interface
* Privacy-focused design

## Technologies Used

* Python
* Tkinter (GUI)
* SQLite
* Cryptography Libraries

## Installation

1. Clone the repository:

```bash
git clone https://github.com/gg71e/secure-password-vault.git
```

2. Navigate to the project folder:

```bash
cd secure-password-vault
```

3. Install required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python main.py
```

## Security

This application follows a privacy-first approach by storing data locally on the user's device. No passwords are transmitted to external servers.
